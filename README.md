# Sodium Sorter

### Smart File Organization, Automatically

**Sodium Sorter** keeps your folders clean without any effort. It continuously monitors your selected directory (Downloads by default) and automatically sorts files into organized subfolders based on their file extensions.

No more cluttered folders. No more wasted time searching for files. Just a clean, organized workspace.

## Features

### Intelligent File Sorting

Automatically moves files into designated folders based on extension rules.

### Custom Rule System

Create, edit, and manage your own sorting rules. Assign any file extension to any destination folder.

### Real-Time Monitoring

Run Sodium Sorter in the background and automatically organize files every 5 seconds.

### Manual Sorting Mode

Sort all existing files instantly with a single click.

### Automatic Folder Creation

Optionally create missing destination folders automatically.

### Modern Glassmorphism Interface

A clean, professional, and visually appealing user experience designed for productivity.

### Portable Application

Distributed as a lightweight standalone executable. No installation required.

### Privacy First

All processing happens locally on your device. No cloud services, no tracking, and no internet connection required.
