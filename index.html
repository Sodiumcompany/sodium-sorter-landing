Why Sodium Sorter?

Stop hunting for files. Sodium Sorter watches your chosen folder (default: Downloads) and moves files into subfolders based on their extension. No more clutter — just clean, organised chaos.

Full control. Add, edit, or delete rules anytime. Choose whether missing folders should be created automatically. Run a one‑time sort or let it run in the background every 5 seconds.

  • Custom rules — map any extension to any folder
  • Live or manual — continuous background sorting or run once
  • Auto‑create folders — toggle on/off with one click
  • Clean glass UI — dark, modern, easy on the eyes
  • Portable — single .exe, no installation required
  • Privacy first — everything stays local, no internet needed
